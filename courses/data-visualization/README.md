# Data Visualization with Python using Plotly

This directory contains notes, code samples, and practical exercises from the Udemy course **Data Visualization with Python using Plotly**. The course focuses on building interactive and dynamic visualizations with the Plotly library in Python.

## Course Overview

This course teaches how to create interactive and professional-quality data visualizations using the Plotly library in Python. It includes practical examples with real datasets and covers both foundational concepts and common visualization techniques.

## What You Will Learn

Throughout this course, you will learn how to:

- Use Plotly to build dynamic and interactive charts
- Create basic and advanced visualization types
  - Line charts
  - Bar charts
  - Scatter plots
  - Bubble charts
  - Pie charts
  - Boxplots
  - Histograms and distribution plots
  - Finance and time series visualizations
  - Subplots
  - Manipulate data using Pandas
  - Work with NumPy for numerical operations
  - Understand when to use each visualization type in data projects
