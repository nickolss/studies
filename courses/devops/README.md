# DevOps: Hands-On Automation (Udemy)

This directory contains notes, code examples, and exercises from the Udemy course **DevOps: Automação sem Enrolação**, which focuses on real-world DevOps automation using modern tools and platforms. The course teaches practical automation skills with strong emphasis on infrastructure provisioning, CI/CD, container orchestration, and monitoring.

## Course Summary

This course covers practical DevOps fundamentals and advanced automation workflows, including:

- Infrastructure provisioning and automation with Terraform on AWS and Azure
- Containerization with Docker and orchestration with Kubernetes (AKS)
- Infrastructure as Code (IaC) and configuration automation with Ansible
- CI/CD pipelines using GitHub Actions with real build, test, deploy, and infrastructure teardown stages
- Monitoring cloud environments using Datadog (metrics, logs, dashboards)
- Scripting and automation with Python for DevOps tasks
- Practical labs to build multi-cloud environments and end-to-end DevOps workflows
